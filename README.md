# 📊 BuzzBuildSystem – Social Media Hype Manager

BuzzBuildSystem is a real-time social media analytics and PR management system built using Java and Data Structures & Algorithms (DSA).  
It simulates how agencies track trending topics, manage incoming messages, and respond to social media engagement efficiently.

---

## 📌 Project Overview

In today’s digital world, tracking social media trends in real time is critical.  
This system processes live messages, extracts hashtags, and identifies trending topics dynamically.

It helps:

- 📈 Monitor real-time social media trends  
- 🔍 Identify top hashtags  
- 🧠 Automate PR workflow  
- 📊 Analyze engagement patterns  

---

## 🧱 System Design

The system is built using **DSA-driven architecture** to ensure high performance and scalability.

- Message ingestion system

- Trend ranking system  
- PR agency assignment system  
- Analytics dashboard

## 🧠 Data Structures Used

This project heavily uses DSA concepts:

- **HashMap** → Track hashtag frequencies  
- **Priority Queue** → Rank trending topics  
- **Queue (FIFO)** → Manage PR task assignment  
- **Stack (LIFO)** → Undo operations  
- **LinkedList** → Store message logs  

(DSA usage explained in PPT page 5)

## 📈 Workflow

As shown in your PPT 

1. User posts message  
2. Hashtags are extracted  
3. Frequency updated using HashMap  
4. Trends ranked using Priority Queue  
5. Message assigned to PR agency (Queue)
